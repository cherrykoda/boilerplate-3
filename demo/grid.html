Grid

GRID allows for the creation of column-based grid that can wrap or nowrap. It is lighter than the FEATURES or TILES mixins in that its contents have no default styling.

Emmet shorthand

.grid>(.grid-cell>p)*3

Mixin usage

                                
div {
    @include grid($align: stretch, $columns: 3, $flow: row wrap, $gutter: 1rem, $justify: flex-start);
}
